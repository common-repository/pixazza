== Description ==

This plugin adds the required javascript for Pixazza (Luminate) webapp and allows to set the parameters with a simply interface.

Using this plugin you can change your theme or upgrade it without set everytime Pixazza code (javascript).

For more information about Pixazza (Luminate) visit:
[www.luminate.com](http://www.luminate.com/)

If you haven't ever seen Pixazza you can see it in over the biggest images at [Art & Artworks](http://www.artartworks.com/).

[Donate to Fabrizio Pivari](http://www.pivari.com/wordpress-plugins/pixazza-wp-plugin/)

== Installation ==

1. Upload `pixazza` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize the plugin in the settings page (Admin > Settings > Pixazza).
   You have to add your site Pixazza ID

== Frequently Asked Questions ==

= I've installed the plugin and activate it but it doesn't seem to work =

Remember you have to create an account for Pixazza [www.luminate.com](http://www.luminate.com/), add your site and manage/configure it then the plugin will work
